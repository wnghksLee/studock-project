# 📚 Studock

과목별 타이머와 학습 계획을 관리하는 Python GUI 앱입니다.

## 설치 방법

```bash
pip install customtkinter
```

## 실행 방법

```bash
python main.py
```

## 기능 요약
* 과목별로 하루 할당량을 지정할 수 있어서 계획 짜기에 용이함
* 과목별로 따로 타이머가 돌도록 설정해둬서 하루에 얼마나 목표를 이뤘는지, 어떤 과목에 많은 시간을 투자했는지 확인할 수 있음

## 세부 기능

| 기능 | 설명 |
|---|---|
|  과목 추가 | 과목명 + 목표 시간(분) 설정 |
|  타이머 시작/일시정지 | 과목별 독립적인 타이머 |
|  초기화 | 해당 과목 타이머 리셋 |
|  목표 수정 | 목표 시간 변경 |
|  과목 삭제 | 과목 제거 |
|  전체 중지 | 모든 타이머 일시정지 |
|  오늘의 통계 | 총 학습 시간 및 과목 수 |
|  테마 전환 | 다크/라이트 모드 |
